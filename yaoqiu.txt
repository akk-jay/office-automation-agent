 办公自动化Agent    个人项目 
项目背景：针对日常办公中重复性操作多、跨系统数据流转繁琐的痛点，设计并开发基于大模型的办公自动化 Agent。用户通过自然语言下达指令（如“整理本
周销售数据并生成周报发送给经理”），Agent 自动完成数据读取、分析、文档生成与邮件发送的全流程闭环。 
负责工作： 
• 使用 LangChain + Function Calling 定义工具函数集（Excel 读写、文件分类整理、邮件发送、日程查询），LLM 根据用户意图自动选择并调用对应工
具，实现自然语言到具体操作的自动映射 
• 基于 LangGraph 构建任务拆解图（StateGraph），将复杂指令拆解为“意图解析→数据获取→分析处理→结果输出→消息推送”等有序步骤节点，支持条
件分支判断与异常回退处理 
• 编写 Python 脚本实现各工具函数的底层逻辑，包括 openpyxl 操作 Excel 数据提取与写入、smtplib 实现邮件自动发送、os/shutil 完成文件系统操作等
